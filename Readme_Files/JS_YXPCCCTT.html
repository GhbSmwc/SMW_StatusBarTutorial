Hex and binary converter

You may be wondering, why did I include this rather simple tool? Well I made this because normally, if a user wants to edit tiles, they have to use a hex editor or a debugger, convert the number to binary (to know what each flags are), tweak some bits, convert back to hex, and write it to ROM/RAM to edit the tile properties. This tool will immediately convert it for you.

Hex: $ = Binary: %
